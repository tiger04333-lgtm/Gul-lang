🦪 Gul-lang (굴랭)

“이 언어를 굴려보자.”

Gul-lang(굴랭)은 한국의 굴찜 포차(Oyster Bar) 문화에서 영감을 받아 탄생한 난해한 프로그래밍 언어(Esoteric Programming Language)입니다.
주문을 외치듯 코드를 작성하며, 굴을 찌는 소리(구…울)와 찜통을 여는 소리(찜)로 프로그램을 제어합니다.

🔥 특징 (Features)

포차 감성: 따뜻한 포장마차 콘셉트의 분위기를 담았습니다.

장음 시스템: 명령어의 길이(정성)가 곧 숫자(Value)가 됩니다.

연속 서빙: 출력(찜)은 줄바꿈 없이 접시에 계속 담깁니다.

📜 사용법 (Usage)

실행 (Run)

python gullang.py example.gul


※ 본 프로젝트는 특정 개인, 단체, 콘텐츠 IP와 무관한 창작 프로젝트입니다.
