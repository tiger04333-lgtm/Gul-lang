🦪 Gul-lang (굴랭)

> "이 언어를 "굴"리자."

Gul-lang(굴랭)은 한국의 굴찜 포차(Oyster Bar)에서 영감을 받아 탄생한 난해한 프로그래밍 언어(Esoteric Programming Language)입니다.  
주문을 외치듯 코드를 작성하며, 굴을 찌는 소리(`구...울`)와 찜통을 여는 소리(`찜`)로 프로그램을 제어합니다.

🔥 특징 (Features)

* 포차 감성: `이세계아이돌`로 개업하고 `포차`로 마감합니다.
* 장음 시스템: 명령어의 길이(정성)가 곧 숫자(Value)가 됩니다.
* 연속 서빙: 출력(`찜`)은 줄바꿈 없이 접시에 계속 담깁니다.
